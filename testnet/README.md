### 1. Installation

#### 1.1. Create & update ```.env``` file
```
HOST=
PORT=
ETHEREUM_URL=
ETHEREUM_MNEMONIC=
DB_HOST=
DB_USER=
DB_PASSWORD=
DB_NAME=
```

#### 1.2. Compile contract

Running ```yarn compile```

#### 1.3. Deploy contract

Running ```yarn deploy```
